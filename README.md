# Elite home ware — GitHub Pages Card

## رفع وتشغيل (بدون تعقيد)
1. أنشئ Repository جديد على GitHub.
2. ارفع محتويات هذا الملف ZIP داخل الريبو (index.html + styles.css + assets).
3. افتح: Settings → Pages
4. اختر:
   - Branch: main
   - Folder: /(root)
5. اضغط Save — سيظهر رابط موقعك.

## تعديل البيانات
- الأسماء والأرقام والروابط موجودة داخل `index.html`.
- الألوان والتصميم داخل `styles.css`.
