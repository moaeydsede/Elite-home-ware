# Elite home ware (GitHub Pages)

## الملفات
- `index.html`
- `styles.css`
- `app.js`

## الرفع على GitHub Pages
1) ارفع الملفات الثلاثة في **جذر الريبو** (بدون مجلدات).
2) Settings → Pages
3) Branch: `main` + Folder: `/(root)` ثم Save

## تعديل البيانات
- الأسماء/الأرقام/الروابط: داخل `index.html`
- الألوان والتنسيق: داخل `styles.css`
